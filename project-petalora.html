Petalora

Стильный и адаптивный лендинг для цветочного магазина

Live Preview GitHub Repo

О проекте

Petalora — это веб-сайт, разработанный для демонстрации ассортимента цветочного магазина. Главной задачей было создать чистый, эстетичный интерфейс, который бы не отвлекал внимание от самих букетов.

Технологии и процесс

  • Верстка: HTML5, CSS3, Flexbox/Grid для адаптивности.
  • Дизайн: Перенос макета из Figma с точным соблюдением отступов и типографики.
  • Деплой: Проект развернут на Vercel через GitHub, что обеспечивает быструю загрузку и CI/CD интеграцию.

Особенности

Особое внимание было уделено мобильной версии сайта, так как большая часть пользователей просматривает магазины со смартфонов. Также реализованы интерактивные hover-эффекты для карточек товаров.
